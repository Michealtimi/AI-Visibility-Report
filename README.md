# AI-Visibility-Report

## Responsive Layout & AdSense Integration

This project is fully responsive for all screen sizes (mobile, tablet, desktop). The main layout uses a flexbox structure:

- **Left AdSense slot:** Visible on medium and larger screens (hidden on mobile)
- **Main content:** Always visible, centered
- **Right AdSense slot:** Visible on medium and larger screens (hidden on mobile)
- **Mobile AdSense slot:** Visible only on small screens, stacked below the main content

All AdSense slots use the `components/adsense-slot.tsx` component. Replace the placeholder AdSense client and slot IDs with your own.

### Customization

- Adjust breakpoints or slot visibility by editing the layout in `app/layout.tsx`.
- For further responsive tweaks, use Tailwind CSS classes in your components.

### Testing

- Test the site on various devices and screen sizes to ensure a seamless experience.
